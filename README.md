# React-pagination


> A Reactjs project

## Build Setup

``` bash
# Lancer MongoDB :
mongod

# Installer les dependances :
npm install

# Lancer le serveur CRUD :
cd TP_ROUTAGE_EXPRESS_CRUD_AJAX
node serverCrud.js

# Lancer l'application Reactjs :
cd my-app
npm start

# Enfin, accéder à l'application sur l'adresse http://localhost:3000/

```
